# **App Name**: VTECHBRIDGE

## Core Features:

- User Authentication: User authentication and authorization via login to access the test.
- Admin Dashboard: Admin dashboard to manage exams and user data.
- Automated questionset chunking: Splits a questionset excel file into multiple parts, e.g., 1-20, 20-40, etc., and each person receives one set.
- Real-Time Timer: Real-time timer during the test to enforce time limits.
- Single Question Display: Display question in the screen one at a time
- Automatic Submission: Automatic submission of test when time is up
- Test Results Display: Display User test result for Admin and users.

## Style Guidelines:

- Primary color: #577399 (RGB), a muted blue, to convey professionalism and trust.
- Background color: #F2F4F7 (RGB), a light, desaturated blue-gray, to provide a clean and calm testing environment.
- Accent color: #FE5F55 (RGB), a vibrant red-orange, to draw attention to critical elements like the timer or submission button.
- Headline font: 'Space Grotesk', a sans-serif with a contemporary, slightly techy, and legible feel; good for a tech-focused app.
- Body font: 'Inter', a sans-serif with a neutral and readable quality; this supports longer chunks of text well.
- Use clear and professional icons for navigation and actions. Icons should be related to testing, time, and results.
- Use subtle animations for loading states and transitions to improve user experience without being distracting.
